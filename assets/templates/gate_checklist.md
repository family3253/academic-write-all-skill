# Gate Checklist

## Gate 1 — Screening Readiness

- [ ] candidate pool populated
- [ ] duplicate handling recorded
- [ ] title/abstract screening started or complete
- [ ] exclusion reasons recoverable
- [ ] source databases listed clearly

## Gate 2 — Evidence Readiness

- [ ] full-text decisions recorded
- [ ] evidence extraction rows present
- [ ] contradiction points preserved
- [ ] claim mapping started
- [ ] weak evidence areas clearly marked

## Gate 3 — Debate / Decision Readiness

- [ ] decision packet built from current artifacts
- [ ] proceed / refine / pivot arguments generated
- [ ] decision record written
- [ ] critical pivot issues explicitly surfaced

## Gate 4 — Writing Readiness

- [ ] section/chapter targets are explicit
- [ ] draftable evidence exists for requested claims
- [ ] results prose is anchored to visible artifacts
- [ ] unresolved gaps are marked, not hidden
- [ ] deliverables manifest is updated

## Gate 5 — Sentinel / Final Handoff

- [ ] sentinel watch report generated
- [ ] citation authenticity report generated
- [ ] gate failures are reconciled with handoff status
- [ ] manifest paths match actual files
- [ ] no unsafe `ready_for_handoff: yes` remains
